Analyzing Data

Data analysis is one of my favorite applications of machine learning. I am experienced with multivariate linear regression, etc.

multivariate Analysis

One of the most important uses of data is through analysis of multivariate models. In this jupyter notebook, I analyzed the features of the data to determine the risk factors of diabetes.

By Analyzing the p value and r^2 value, we can determine which features correlate with diabetes.

by Analyzing the combined variable model, we can see the relative importance of the features by their p values. This model had a combined r^2 value of 0.5, meaning that 50% of the variance in diabetes levels could be explained by these features.
